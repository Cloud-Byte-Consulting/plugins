The workflows in this plugin were prepared for publication by Cloud Byte Consulting from its
organization-managed Document Hub records. Each `SKILL.md` retains a record link so maintainers can
trace the business request that produced it; those records may require organization access.

The repository's Apache-2.0 license applies to the skill implementations published here. A source
link is provenance, not a claim that unrelated third-party material is relicensed.

Two Document Hub records referenced external prompt packages: `consumer-agent-landscape` and
`life-engine-setup`. Their published `SKILL.md` files are original Cloud Byte Consulting
implementations of the underlying business requests. They do not reproduce the third-party prompt
text.

Before adding another externally sourced workflow:

1. Confirm that Cloud Byte Consulting owns it or has publication and relicensing permission.
2. Preserve any required copyright and license notices.
3. Rewrite from functional requirements when permission does not cover the original expression.
4. Record the decision in this file and retain the source link in the skill.
